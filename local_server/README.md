# Brother QL プリンタープロキシサーバー

## 概要
このアプリケーションは、ローカルネットワーク内のBrother QLプリンターをインターネット経由でアクセス可能にするプロキシサーバーです。

## セットアップ

1. **初回セットアップ**
   ```bash
   setup.bat
   ```

2. **開発環境での実行**
   ```bash
   run.bat
   ```

3. **EXEファイルのビルド**
   ```bash
   build.bat
   ```

## 使用方法

### 1. プロキシサーバーの起動
- EXEファイルを実行するか、`run.bat`を実行
- システムトレイにアイコンが表示されます

### 2. 設定
- 設定タブでプリンターのIPアドレスを入力
- ngrokを使用する場合は、認証トークンを入力

### 3. サーバーの開始
- 「サーバー開始」ボタンをクリック
- ローカルURLと外部URL（ngrok使用時）が表示されます

### 4. Cloud Functionsの設定
環境変数に以下を設定：
- `USE_PROXY`: "true"
- `PRINTER_PROXY_URL`: プロキシサーバーのURL（ngrokのURLまたは固定IP）

## API エンドポイント

- `GET /` - Webインターフェース
- `GET /status` - ステータス確認
- `POST /print/raw` - 生データ印刷
- `POST /print/label` - ラベル印刷

## トラブルシューティング

### プリンターに接続できない
1. プリンターのIPアドレスが正しいか確認
2. プリンターとPCが同じネットワークにあるか確認
3. ファイアウォールでポート9100がブロックされていないか確認

### ngrokが動作しない
1. ngrok認証トークンが正しく設定されているか確認
2. `pip install pyngrok`でpyngrokがインストールされているか確認
