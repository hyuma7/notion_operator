# Test Files - Notion QR Label Generator

## 📋 概要

テスト用のファイルとサンプル画像を格納しています。

## 📁 ファイル一覧

### **label_*.png** - テスト用画像ファイル
- 実際に生成・印刷テストされた画像
- Brother QLプリンターでの印刷確認済み
- サイズ・フォント調整の参考用

### **requirements.txt** - ローカル開発用依存関係
- PIL (Pillow)
- qrcode[pil]
- その他ローカル開発で必要なライブラリ

## 🎯 用途

- 開発時の参考ファイル
- 印刷品質の確認
- フォントサイズ・レイアウトの検証

## 🗑️ 管理

これらのファイルは：
- **Cloud Functionsにはデプロイされません**
- **開発・テスト用途のみ**
- **定期的にクリーンアップ可能**
